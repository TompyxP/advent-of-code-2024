# Advent of Code 2024

This repository contains my solutions for [Advent of Code 2024](https://adventofcode.com/2024) implemented in TypeScript.

## Setup

```bash
npm install
```

## Running Solutions

Use npm scripts to run individual day solutions:

```bash
npm run day1    # Run day 1 solution
npm run day2    # Run day 2 solution
# ... and so on
```
